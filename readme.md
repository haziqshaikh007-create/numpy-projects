# 📊 Python Data Analysis Projects

This repository contains a collection of Python projects focused on **data analysis, logic building, and practical problem-solving**. Each project demonstrates the use of Python libraries such as **NumPy, Pandas, and Matplotlib**, along with algorithmic thinking to analyze and interpret datasets.

---

## 🚀 Projects Included

### 1. Students Marks Analyzer
- Reads student marks data and computes performance statistics.
- Features:
  - Average, median, and highest/lowest marks.
  - Subject-wise performance comparison.
  - Simple visualization of results.

### 2. Sales Data Analysis
- Analyzes sales records to identify trends and insights.
- Features:
  - Monthly and yearly sales summaries.
  - Top-performing products and categories.
  - Revenue growth visualization.

### 3. Random Dataset Analyzer
- Works with randomly generated datasets for practice.
- Features:
  - Data cleaning and preprocessing.
  - Statistical summaries (mean, variance, correlations).
  - Pattern detection and anomaly identification.

### 4. Recommendation System Logic
- Implements the basic logic of a recommendation system.
- Features:
  - User-item interaction matrix.
  - Similarity-based recommendations.
  - Prototype for collaborative filtering.
